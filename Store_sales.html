Problem

Corporación Favorita is a large Ecuadorian-based grocery retailer. Corporación Favorita hope they could predict the number of sales in each store with given data to prevent overstocking and increase customer satisfaction. Therefore, we will build a predictive model to help the company predict their future sales. Corporación Favorita will evaluate the models we build by Root Mean Squared Log Error (RMSLE). The model with lowest RMSLE will be accepted by the company.

Data Overview

The company offers several datasets for us to build a predictive model. (We get these data on Kaggle)

  • Oil data: This dataset includes the daily price of oil from 2013-01-01 to 2017- 08-31.
  • Holiday events: This dataset includes all kinds of holidays from 2012-03-02 to 2017-12-26.
  • Train: This data set includes data with the sales of each product in different stores from 2013/01/01 to 2017/08/15.
  • Test: This dataset includes data same as train dataset, but it doesn't have sales of each product. Our goal is to predict the sales of every product in this dataset. The data in test set ranges from 2017/08/16 to 2017/08/31.
  • Stores: The information of each stores including city, state, type, and cluster.
